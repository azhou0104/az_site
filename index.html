Event Discovery Mobile App

As an independent contractor, I helped build an Event Discovery mobile app (think Eventbrite or Meetup) that ranks and recommends events based on the users' locations, preferences, and friends. Users can create profiles to view and participate in events, and organizers can create profiles to market their hosted events.

The app also implements the Ticketmaster API to retrieve larger-scale events.

Built with Flutter, Dart, Typescript, Firebase cloud functions, and Firestore.
